# 关于导航零号

导航零号目前还是半保密的项目，不适合过多的介绍。

导航零号基于自研的SDK开发。具有完整的算路、搜索、路况、引导等全部导航功能。

我们期望将来能做成全开源的项目，服务于所有需要导航技术的公司企业。开源计划还在酝酿中。
